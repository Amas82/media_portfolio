I learned how to lay out the foundation of a website and work with CSS, JavaScript and jQuery. To create the forms, you need JavaScript form validation for the key fields in the contact form. I have taken advantage of adding the social media on the dashboard. The social media peek-out sidebar at the left edge of the browser window stays hidden until the user hovers over it which then reveals the various social media icons.

I created an advertisement that relates to the topic ‘Save Our Ocean’. Below the image I created three columns to read as an article.

Goal: My title is "Save the Ocean", which is short and gives attention to the concept. This course project was to learn some new things about layout, CSS timing, CSS responses, and JavaScript interactivity. jQuery is a JavaScript library that includes pre-built functions that makes it easier to add interactivity, animations, and other effects, such as hover effects to the page without adding a lot of JavaScript code. Media queries work by specifying certain conditions and applying a specific stylesheet to them. Media queries are used in responsive web design, which allows for a site to be viewed on a variety of screen sizes and browsers.

Process used: My website shows a proper contact page and error message. To create the forms, you need JavaScript form validation for the key fields in the contact form. I have taken advantage of adding the social media on the dashboard. The social media peek-out sidebar at the left edge of the browser window stays hidden until the user hovers over it which then reveals the various social media icons. When the users hover their mouse over the various social media slidebar it animates the various social media icons. In my website I added four images to put in the sliders. The images move from left or right in the sliders. There is a brief description for each of the images in the hide and show captions. Artifact: Divine Lip Bliss

The navigation menu bar is not active, but the site is responsive across different device sizes.

Goal: I had to create a responsive website for a fictional product that can be viewed across different device sizes; using media queries to control the layout for desktop, tablet, and phone. I had to create a visual idea for the fictional product through a name, product features, benefits, and target market. Also keeping in mind, the value proposition or features that the fictional product had to be unique.

Process used: This was the hero website which features a fictional product. The company mission statement and services have links provided on the website. The elements of the website collectively catch the user's eye through the structure, functionality, and website visuality. There are key elements of an effective way to create and design the website. The navigation gives a hover effect with the products and ‘about us’ links and links within a page’s content to help users jump directly to a page to find the information about the company's mission statement, original logo for the company, and appropriate images and colors to matches the product. Also, the hover effect changes it colors when the mouse hovers over the menu links. I added the @media query and I included the hover drop down navigation menu for a subhead to the fictional product website. The selection of typeface is readable and tells a story and the hero image is relevant to the site. I briefly summarized the product features, benefits, target market and what makes the product unique.
